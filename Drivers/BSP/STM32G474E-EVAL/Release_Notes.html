Release Notes for STM32G474E-EVAL BSP

Copyright Â© 2019 STMicroelectronics

Purpose

This driver provides a set of functions to manage peripherals mounted on Evaluation Board :

  • LEDs and push-button

  • LCD Module

  • Audio

  • SD Card

  • QSPI Memory

  • Smart Card

  • Temperature sensor

  • Idd measurement

  • UCPD Power measurement
